Data Manipulation Language(DML) :

The main purpose of this DML commands or operations are to perform any modifications on the data or table using the INSERT, UPDATE, DELETE.

INSERT :

This command is used to insert a new data into the database tables

Syntax :

INSERT INTO < TABLE NAME > (COLUMNNAME1, COLUMNAME2...) VALUES(VALUE1, VALUE2...)



UPDATE :

This command is used to update data in database tables.

Syntax :

UPDATE < TABLE NAME > SET < COLUMN NAME> = VALUE WHERE condition



DELETE :

This command is used to delete data in database tables.

Syntax :

DELETE FROM < TABLE NAME > WHERE condition



Data Query Language (DQL) :

The main purpose of this DQL commands is to perform a data retrive operations on the database tables, SELECT

Syntax :

SELECT COLUMNNAME FROM < TABLE NAME > WHERE Conditional (optional)
